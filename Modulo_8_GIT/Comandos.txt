git init
Inicia repositorio

git status
Verifica status

git add .
Adiciona arquivos no stage

git commit -m ""
Grava alterações no stage

git push
Enviar alterações para repositório remoto

git log
Ver o histórico

git checkout ID
navegar entre versões

git log --all
ver todas as versões

git checkout master
Navega até branch principal

git checkout -b "nome_da_branch"
cria e muda para a nova branch

git checkout "nome_da_branch"
Muda para a branch

git branch
Mostra as branchs

git merge nome_da_branch
mescla a branch atual com a branch indicada

TRABALHANDO EM EQUIPE

Navega até o repositório e clica em fork
usa o git clone para baixar os arquivos para o repositório local
cria uma nova branch (git checkout -b "teste")
faz as modificações
faz o push para a nova branch (git push origin teste)
Depois vai ni GitHub e procura a opção contribuir
